About

DappsList.net is a set of directories of dApps for different chains, live and ready to be used by end users.

We've created this set of lists because official ecosystem pages usually list a lot of non-live projects, and and plenty of projects are not intended for end users. Which makes it hard to find cool dApps on official ecosystem pages.
